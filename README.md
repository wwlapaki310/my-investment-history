# 私の投資ヒストリー 📈

投資の履歴（資産推移）を可視化・投稿・共有するWebアプリケーション

## 📝 概要

「私の投資ヒストリー」は、個人投資家が自身の資産推移を記録し、可視化できるWebアプリケーションです。
年ごとの資産額を入力するだけで、美しいグラフで投資の軌跡を振り返ることができます。

### 主な特徴

- ✨ **簡単入力**: Web上で年と資産額を入力するだけ
- 📊 **リアルタイムプレビュー**: 入力と同時にグラフが更新
- 💾 **自動保存**: ブラウザに自動保存（サーバー不要）
- 📤 **データ共有**: JSON形式でエクスポート/インポート可能
- 💬 **コメント機能**: 各年の出来事や感想を記録
- 📱 **レスポンシブ対応**: スマホでも快適に利用可能
- 🆓 **完全無料**: サーバー費用不要、広告なし

## 🚀 使い方

1. **データ入力**
   - 年と資産額を入力フォームに記入
   - コメント欄に投資方針や出来事を記録（任意）
   - 「追加」ボタンをクリック

2. **グラフ確認**
   - 入力データが自動的にグラフ化
   - 資産推移を視覚的に確認

3. **データ管理**
   - 編集・削除機能で柔軟に管理
   - JSONエクスポートでバックアップ
   - 他のデバイスではインポートで復元

## 🛠️ 技術スタック

- **フロントエンド**: HTML5 / CSS3 / JavaScript (Vanilla)
- **グラフ描画**: Chart.js
- **データ保存**: localStorage (ブラウザ内蔵)
- **ホスティング**: GitHub Pages
- **費用**: 完全無料
