# 私の投資ヒストリー

投資の履歴(資産推移)を可視化・投稿・共有するWebアプリケーション

## 🎯 コンセプト

投資家が自身の資産推移をグラフで可視化し、コメントとともに投稿・共有できるプラットフォームです。
- 横軸:時間(年・月)
- 縦軸:資産額
- グラフ下部:コメント・振り返り

## ✨ 主な機能

### MVP(最小機能)
- 📊 **簡単データ入力**
  - CSVファイルアップロード対応
  - 手入力フォーム(日付・金額のみ)
  - テンプレートからコピー
- 📈 **グラフ自動生成**
  - 折れ線グラフで資産推移を可視化
  - インタラクティブな表示(Chart.js使用)
- 💬 **コメント投稿**
  - グラフに対する振り返り・学び
- 👀 **閲覧・共有**
  - 他のユーザーの投資履歴を閲覧
  - URLで簡単共有

### 将来的な機能
- 匿名/実名選択
- いいね・コメント機能
- 投資手法タグ付け
- 統計情報(平均リターン等)
- エクスポート機能

## 🚀 技術スタック

- **フロントエンド**: HTML/CSS/JavaScript(純粋、フレームワーク不要)
- **グラフ描画**: Chart.js
- **データベース**: Firebase Firestore
- **認証**: Firebase Authentication(Googleログイン等)
- **ホスティング**: GitHub Pages / Vercel

詳細は [ARCHITECTURE.md](ARCHITECTURE.md) を参照

## 🔧 開発環境のセットアップ

```bash
# リポジトリをクローン
git clone https://github.com/wwlapaki310/my-investment-history.git
cd my-investment-history

# ローカルサーバーで起動(例:Python)
python -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```
